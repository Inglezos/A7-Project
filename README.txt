Change the mobile number inside A7_Thinker.ino file with the desired one.

To start the system, call the number of the SIM card inserted into the module, wait for the module to hang up and to call you back. Then, you hang up and that's when the system starts receiving satellite GPS data. This "handshake" is necessary.

The mobile number of the existing SIM card is 6942234642. Make sure there is an active mobile carrier package for calls and SMS, otherwise the system will malfunction.
